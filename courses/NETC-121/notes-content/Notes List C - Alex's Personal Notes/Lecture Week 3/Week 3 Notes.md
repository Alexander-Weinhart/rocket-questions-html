# Week 3 Notes

## Ethernet and TCP/IP

| TCP/IP Model | OSI Model |
|---|---|
| Application Layer | Application Layer |
| Application Layer | Presentation Layer |
| Application Layer | Session Layer |
| Transport Layer | Transport Layer |
| Internet Layer | Network Layer |
| Network Access Layer | Data Link Layer |
| Network Access Layer | Physical Layer |

## ARP and RARP

- ARP sends out IP address and requests MAC address.
- RARP sends out MAC address and requests IP address.

## MAC Address Scope

- MAC address information ends at the router.
- MAC addressing cannot go between subnets.

## Traffic Types

- Unicast
- Multicast
- Broadcast: broadcast MAC address is `FFFF.FFFF.FFFF`.

## ICMP

- `ping`
- `tracert`
- `traceroute` (Linux)
