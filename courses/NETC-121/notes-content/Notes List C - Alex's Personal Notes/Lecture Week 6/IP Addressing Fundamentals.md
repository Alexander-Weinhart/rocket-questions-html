# IP Addressing Fundamentals

## IPv4 Basics

- IPv4 addresses are 32 bits total.
- They are written as four 8-bit octets (for example: `192.168.1.10`).
- The full set of possible addresses is the IPv4 address space.
- Total theoretical IPv4 addresses: about 4.29 billion (`2^32`).

## Octets and Binary

- Each octet ranges from `0` to `255`.
- Each octet is 8 bits, so each bit position has a place value.
- Binary in each octet is converted to decimal for readable IPv4 notation.

## IPv6 Basics

- IPv6 addresses are 128 bits.
- This is a much larger address space than IPv4.
- Approximate total IPv6 addresses: `3.4 x 10^38` (`2^128`).
- IPv6 exists because IPv4 address space is limited.
