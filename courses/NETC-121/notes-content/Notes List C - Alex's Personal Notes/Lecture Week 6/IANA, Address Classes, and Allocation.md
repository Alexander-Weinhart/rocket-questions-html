# IANA, Address Classes, and Allocation

## What IANA Is

- IANA is the Internet Assigned Numbers Authority.
- It manages global IP address resources and protocol numbering coordination.

## Trickle-Down Allocation Model

- IANA allocates large IP blocks to Regional Internet Registries (RIRs).
- RIRs allocate to ISPs, enterprises, and organizations.
- Providers then assign addresses to customers and networks.
- This top-down process is often described as a trickle-down block allocation model.

## Classful IPv4 Ranges (Historical)

- Class A: very large networks, about 16 million host addresses per classful network.
- Class B: medium networks, about 65,000 host addresses per classful network.
- Class C: small networks, up to 256 total addresses (254 usable hosts) per classful network.

## IPv4 Exhaustion Context

- Public IPv4 pools have been largely allocated at global/RIR levels.
- Because IPv4 is limited, NAT and private addressing became critical for scale.
- IPv6 deployment addresses long-term growth needs.
