# Multiplexing

## What Multiplexing Is

- Multiplexing is sending multiple signals across the same physical line at the same time.
- It improves link efficiency by sharing one medium among multiple data streams.

## Why Multiplexing Is Used

- Reduces cost by avoiding separate dedicated links for each signal.
- Increases utilization of available bandwidth.
- Supports many users/services over shared infrastructure.

## Frequency-Division Multiplexing (FDM)

- Different signals use different frequency bands on the same link.
- All channels can be active simultaneously.
- Each channel stays in its assigned frequency range.
- Common in radio/cable-like channelized systems.

## Time-Division Multiplexing (TDM)

- Different signals share the same link by taking turns in time slots.
- Each source transmits during its assigned slot.
- Appears simultaneous at system level because switching is very fast.
- Works well when multiple streams need predictable access.

## Core Contrast: FDM vs TDM

- FDM splits by frequency.
- TDM splits by time.
- Both allow multiple logical channels over one physical path.
