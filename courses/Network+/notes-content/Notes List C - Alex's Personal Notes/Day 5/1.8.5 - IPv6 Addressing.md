## IPv6 Compression
  Uncompressed - 2600:DDDD:1111:0001:0000:0000:0000:0000:0001
    Remove the leading zeros
      2600:DDDD:1111:1:0:0:0:1
      Get rid of the 2+ groups of zeros with double colons
      2600:DDDD:1111:1::1

## IPv4 to IPv6 Communication
  Tunneling - encapsulate one protocol within another
  6to4 addressing
  Send IPv6 over an existing IPv4 network
  Creates an IPv6 address based on the IPv4 address
  Requires relay routers
  No support for NAT
  No longer available as an option on Windows
  4in6 tunneling
  Tunnel IPv4 traffic on an IPv6 network
  Dual-stack - have the option to use both IPv4 and IPv6
  Run both at the same time
      
  Translate - convert between IPv4 and IPv6
  NAT64 router and DNS64 server
  Requires something in the middle to translate, that NAT64 router
