## Three Tiered Architecture
  Core:
    Central point of the network
    Web servers, databases, applications
  Distibution:
    Midpoint between core and users
  Access:
    Where users connect
    End stations, printers
## Collapsed Core Architecture
  Core and distribution layer are collapsed
## Traffic flows
  East-west:
    Traffic between devices in the same data center
    Reliably fast response times
  North-south:
    Ingress-egress to an outside device
    Not in the data center
