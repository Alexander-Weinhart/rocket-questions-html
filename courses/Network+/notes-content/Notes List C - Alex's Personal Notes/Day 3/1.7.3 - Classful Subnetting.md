## Classes
| Class | Leading Bits | Number of Networks | Hosts per Network |
| ----- | -------------- | ------------------ | ----------------- |
| A     | 0xx (0-127)    | 128                | 16,777,214        |
| B     | 10xx (128-191) | 16,384             | 65,534            |
| C     | 110x(192-223)  | 2,097,152          | 254               |
| D     | 1110(224-239)  | not defined        | not defined       |
| E     | 1111(240-255)  | not defined        | not defined       |

## ONE SUBNET CANNOT INCLUDE ALL CLASS ADDRESSES FOR A, B, OR C
## Class D and E
Class D:
  Multicast addresses
  Not assigned to a device
Class E:
  Reserved addresses
  Not used for any purpose
