## Now for the Video's Notes:

The N10-009 Network+ certification is good for three years, If the exam version is expired, there will be a big red header on all the related pages saying "THIS EXAM CONTENT IS EXPIRED, DO NOT STUDY THIS EXAM. PLEASE STAND BY FOR A WEBSITE UPDATE"

That expiration date for this exam is in 2027, more info on that will appear soon right here.

#### The actual exam
 - We're looking at a 90 minute, 90 question exam. The questions you know will take less time to answer anyway.
 - You'll need a passing score of 720 on a scale of 100-900. that's a 77.5% or higher to pass.

| Domain                        | % of exam |
| ----------------------------- | --------- |
| 1.0 - Networking Concepts     | 23%       |
| 2.0 - Network Implementation  | 20%       |
| 3.0 - Network Operations      | 19%       |
| 4.0 - Network Security        | 14%       |
| 5.0 - Network Troubleshooting | 24%       |
