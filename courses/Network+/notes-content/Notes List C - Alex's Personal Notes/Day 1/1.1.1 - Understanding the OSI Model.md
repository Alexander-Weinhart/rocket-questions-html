## The OSI Model

The Open Systems Interconnection model

| Layer # | Name | Purpose | Layer Nickname | Protocols | Referring To |
| ------- | ------------ | ------------------------------------------------------------------------- | ---------------------------- | -------------------------------------- | -------------------------------------------------- |
| 7       | Application  | the layer we see on the screen,                                           | no nickname                  | HTTP, FTP, DNS, POP3                   | your eyeballs                                      |
| 6       | Presentation | character encoding, encryption, combined with application layer.          | no nickname                  | SSL/TLS                                | SSL/TLS                                            |
| 5       | Session      | communication management for the transport layer                          | no nickname                  | control protocols, tunneling protocols |                                                    |
| 4       | Transport    | getting information from one side of WAN To another                       | the post office layer        | TCP, UDP                               | TCP segment, UDP datagram                          |
| 3       | Network      | uses IP addresses to forward traffic between local networks (subnets)     | the routing layer            |                                        | packet                                             |
| 2       | Data Link    | uses MAC addresses to allow the network to target computers by their NICs | MAC Address/ switching layer |                                        | frame. Extended unique Identifier (EUI-48, EUI-64) |
| 1       | Physical     | signals, cabling, and connectors                                          | no nickname                  |                                        |                                                    |
