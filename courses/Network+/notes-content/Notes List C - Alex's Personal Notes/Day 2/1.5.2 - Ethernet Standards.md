
| IEEE Standard | Description | Media | Speed |
| ------------- | ------------------- | ------ | ------ |
| 1000BASE-T    | Gigabit Ethernet    | copper | 1Gbps  |
| 10GBASE-T     | 10 Gigabit Ethernet | copper | 10Gbps |
| 1000BASE-SX   | Gigabit Ethernet    | Fiber  | 1Gbps  |


BASE - Baseband- like broadband for cable
  T is for twisted pair copper
  F is for fiber
  SX would be short wavelength light sent over fiber
