ICMP - internet control message protocol
  - ping
GRE - Generic Routing Encapsulation
  - VPNs
    - encrypts GRE data
  - encapsulates traffic inside of IP
IPSec - Internet Protocol Security
  Security for OSI layer 3
  Confidentiality and integrity/anti-replay
    - encryption and packet signing
  Very standardized
  Two core IPSec protocols
    Authentication Header (AH)
      Used to validate the information over IPSec tunnel
      Sending information in the clear but including additional hashing
    Encapsulation Security Payload (ESP)
      Encrypts the ESP header, the data and the ESP trailer
      Authenticates the ESP header along with IP header, data, and ESP trailer
  Internet Key Exchange (IKE)
    Agree on encryption/decryption keys
      - no sending across network
      - builds a security Association (SA)
    Phase 1:
      Use Diffie-Hellman to create a shared secret key
        Diffie-Hellman:
          -
      UDP/500 - ISAKMP - Internet Security Association and Key Management Protocol
      Building the ISAKMP tunnel
    Phase 2:
      Coordinate Ciphers and key sizes
      Negotiate Inbound and outbound SA for IPsec
      Including the encrypted data over the Encapsulation Security Protocol - ESP tunnel
