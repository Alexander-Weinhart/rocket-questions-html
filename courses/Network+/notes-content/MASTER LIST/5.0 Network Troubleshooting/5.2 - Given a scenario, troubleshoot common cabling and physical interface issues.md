# 5.2 - Cabling and Physical Interface Issues

## Core Idea

Many network problems begin at the physical or interface layer. You must be able to recognize incorrect media, signal problems, port-state issues, and hardware mismatch symptoms.

## Cable Issues

### Incorrect Cable

- Wrong media choice can prevent or degrade connectivity.

#### Single-mode vs. Multimode

- Fiber type mismatch affects optics compatibility and distance expectations.

#### Category 5/6/7/8

- Twisted-pair categories differ in supported performance and intended use.

#### STP vs. UTP

- Shielding choice affects interference resistance and installation requirements.

### Signal Degradation

#### Crosstalk

- Signal bleed between conductors.

#### Interference

- External electromagnetic noise affecting signal quality.

#### Attenuation

- Signal weakening over distance or poor media conditions.

### Improper Termination

- Bad connectorization or punch-down work can break or degrade links.

### TX/RX Transposed

- Incorrect transmit/receive pairing can break communication where auto-correction is absent.

## Interface Issues

### Increasing Interface Counters

- Error counters reveal ongoing link problems.

#### CRC

- Frame integrity errors.

#### Runts

- Frames smaller than valid minimum size.

#### Giants

- Frames larger than expected maximum size.

#### Drops

- Traffic discarded due to congestion or interface issues.

### Port Status

#### Error Disabled

- Port shut down by protective logic after a fault condition.

#### Administratively Down

- Port manually shut down by configuration.

#### Suspended

- Port present but not forwarding normally because of control logic or bundling issues.

## Hardware Issues

### PoE

- Power over Ethernet problems can affect connected devices.

#### Power Budget Exceeded

- Available PoE power is insufficient for all attached demand.

#### Incorrect Standard

- Power negotiation and standards mismatch can prevent normal operation.

### Transceivers

#### Mismatch

- Unsupported or incompatible optics/media combinations can fail.

#### Signal Strength

- Poor optical power levels can cause unstable links.

## Big Picture

When a link behaves badly, think media type, connector quality, signal condition, interface state, counters, and power before jumping straight to higher-layer theories.
