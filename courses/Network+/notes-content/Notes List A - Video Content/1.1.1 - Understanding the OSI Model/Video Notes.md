## Core Ideas

- We'll start our Network+ course with a broad discussion of something we refer to as the OSI model.
- This is referring to the Open Systems Interconnection Reference model, and it's a model we use in it to describe the process that data takes as it traverses our networks.
- The OSI model is not designed to be a detailed description of this data but instead is intended to describe a broad overview of how data traverses our systems.
- We're also not describing the OSI protocol suite.
